Transaction Editor

WYSIWYG Transaction Editor

  • Load existing transactions, UTXOs, or addresses
  • Import PSBTs Coming soon!
  • Create new transactions and outputs
  • Connect existing and custom transactions
  • Duplicate existing transactions as new, editable ones for complex reorg or double spend operations Coming soon!
  • Edit transactions, output scripts, and signature/witness data
  • Sign and broadcast transactions Coming soon!
  • Export PSBTs or raw transactions
  • Your editor state is saved in cookies Coming soon!

This early-access software runs in an untrusted environment, and may contain bugs. Ensure that you verify the content of all transactions in your native wallet software after exporting.
